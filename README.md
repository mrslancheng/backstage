# 前言

该项目一套多功能的后台管理模板，适用于80%的后台管理系统开发。基于VUE,使用Element-UI框架；可快速完成后台管理开发与二次开发。

#功能

Element UI
登录/注销
窗口最大化
案件列表
权限管理
系统日志
用户信息
404
进度条插件
AXIOS
路由守卫

# 安装步骤

``` bash
# 安装依赖
npm install

# 启动vue项目  //可以使用localhost:8080、192.168.1.70:8080两种方式打开
npm run dev

# 项目打包命令
npm run build
